## 🔬 Featured Project

### [🧠 Optical Neural Network](https://github.com/sunshine0900/optical-neural-network)

> MLP model for predicting spectral reflection and transmission coefficients of multilayer thin-film coatings

- **Physics:** Transfer Matrix Method (TMM) for two-layer optical structures (SiO₂, TiO₂, MgF₂)
- **Architecture:** MLP · 9→128→256→128→64→4 · BatchNorm · Dropout · ReLU
- **Training:** 10 epochs on Google Colab (NVIDIA T4 GPU), best model saved by Test Loss
- **Result:** Train Loss ~0.019 · Test Loss ~0.007
